# BIQU-B1SE-Plus-Marlin-2.1.2.5

Marlin 2.1.2.5 firmware Build for the BIQU B1 SE Plus

Flash Marlin to BIQU B1 SE Plus
-Copy firmware.bin to SD card
-Place in machine
-Turn on power

Flash BIGTREETECH User Interface From: BIQU github https://github.com/bigtreetech/BIQU-B1-SE-PLUS/tree/master/B1-SE-PLUS%20firmware/BTT-mode
-Copy firmware.bin and TFT35 folder to SD Card
-Place in machine
-Turn on power NOTE: Remove TFT35 folder from SD Card because will reload each time power is applied.
